Blackjack
======

Just a little Blackjack App to demonstrate BDD.

Stories
------

- Player starts the game
- Player takes a new card
- Player wins
- Player loses
- Player start again
- Player closes the game


Tools
-----

- Ruby 1.9.2
- RSpec
- Mocha
- Cucumber
